Инициализация приложение
Зайти в папку new_prod
Открыть терминал
прописать sudo docker-compose up
После запуска зайти на адрес 127.0.0.1:8000/login
После захода на адрес должно быть предложение о совершении миграции, нажать кнопку запуска миграции.
Пользователи создаются как обычно по адресу /register
Для тестирования (добавление товаров, администраторов, продуктов, изменения баланса) нужно перейти на адрес /create 
Важно предварительно зарегистрироваться и залогиниться под любым произвольным пользователем, иначе не пустит в create.
Пользователи регистрируются в порядке возрастания айди начиная с одного, при необходимости айди пользователя можно посмотреть у админа в списке заказов.
По адресу / (т.е главная страница) находится список товаров и возможность их добавления для авторизованых пользователей.
Для перехода в корзину в правом верхнем углу нужно нажать Personal account
В случае если пользователь имеет права администратора, то после перехода Personal account, у него будет дополнительная вкладка администратора.

! в create при изменении баланса, он меняется у текущего пользователя под которым был выполнен вход.
